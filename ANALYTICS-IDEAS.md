# TON Quant — кросс-аналитика, которая взрывает мозг

Ресёрч: методологии Glassnode (режимы рынка через дивергенции), Nansen (smart money netflow, holder quality), академические работы по wash-trading на DEX (arXiv 2102.07001, ACM 2024), микроструктура рынка (depth/slippage/CVD). Всё адаптировано под наши данные: TONAPI + DexScreener + STON.fi.

Легенда готовности: ✅ — считается из данных, которые уже приходят на страницу; 📸 — нужны ежедневные снапшоты (GitHub Actions, бесплатно); 🔑 — тяжёлый сбор (история транзакций по кошелькам).

---

## I. Дивергенции — главный источник альфы

Сила не в метриках, а в их **расхождениях**. Одна метрика = факт, две разнонаправленные = сигнал.

**1. Матрица «Цена × Холдеры»** 📸 — четыре квадранта:
- холдеры ↑ / цена ↓ → **тихая аккумуляция** (главный buy-сигнал Glassnode-логики)
- холдеры ↑ / цена ↑ → здоровый рост
- холдеры ↓ / цена ↑ → памп на тонком рынке, выходят в силу (exit-сигнал)
- холдеры ↓ / цена ↓ → умирание
Каждый токен топ-20 — точка на этой плоскости. Один взгляд — весь рынок TON по стадиям жизни.

**2. «Ликвидность × Цена» = Rug Radar** 📸 — TVL падает 3+ дня подряд при стабильной цене → ЛП тихо выходит, краш впереди. Обратное (TVL ↑ при флэте) → маркетмейкер готовит движение. Никто на TON это не показывает.

**3. «Объём × Цена» — абсорбция** ✅ — buys >> sells, а цена не растёт → крупный продавец поглощает спрос (стена). Sells >> buys при флэте → кто-то всё откупает. Данные txns уже на странице — нужна только интерпретация и подсветка.

**4. «Активность × Цена» (Nansen DAA-style)** 📸 — число транзакций растёт при флэте цены → интерес опережает прайс; транзакции умирают при росте → последний вздох пампа.

**5. Profit-at-Price (MVRV-lite)** ✅ — честный UTXO-анализ нам недоступен, но рабочая аппроксимация: «% дней за последние 90, когда цена была ниже текущей». 95% → почти все холдеры в плюсе → зона фиксации прибыли. 5% → почти все в минусе → капитуляция/дно. Считается из графика, который уже тянем.

## II. Микроструктура и детекция манипуляций

**6. Wash-Trading Detector** ✅ (база) — три красных флага из академических работ:
- Volume/TVL > 3–5х в сутки при цене ±1% → объём фейковый
- объём на одного мейкера (volume/makers) аномально высокий → крутят между своими
- fee-доход пула не бьётся с заявленным объёмом
Бейдж «🚿 wash?» на токене. Chainalysis оценила wash-объём DEX в $2.57B/год — фича актуальна каждому, кто смотрит на «топ по объёму».

**7. Slippage Map** ✅ — из резервов пула (x·y=k) **точно** считается прайс-импакт ордера $100 / $1K / $10K по каждому пулу. «Хочешь купить на $5K — двинешь цену на 7.3%, лучше частями по $800 через 2 пула». Это конвертирует ликвидность из абстрактной цифры в торговое решение. Чисто клиентская математика.

**8. CVD (кумулятивная дельта объёма)** 📸 — кумулятив (buys−sells) по дням поверх графика цены: классика микроструктуры. CVD растёт, цена нет → давление покупателей зреет.

**9. Спред-скринер всех 20** ✅ — кросс-DEX спреды уже считаем per-token; вывести таблицей по всем сразу = карта арбитражных дыр экосистемы.

## III. Кросс-токенное и портфельное

**10. Корреляционная матрица топ-20 + бета к TON** ✅ — из 7д/30д графиков (уже тянем все 20): тепловая матрица корреляций. Сразу видно: кластеры (мемы ходят вместе?), у кого бета к TON >1.5 (рычаг на рост TON), кто живёт своей жизнью (диверсификатор). На TON такого нет ни у кого.

**11. Lead-Lag радар** 📸 — кросс-корреляция со сдвигом по времени: какой токен двигается первым, кто догоняет с лагом. Если NOT исторически ведёт мем-кластер — движение NOT это сигнал на остальные.

**12. Структура холдеров: шримпы/дельфины/киты** ✅ — из топ-100 холдеров режем supply на корзины (<0.01%, 0.01–0.1%, 0.1–1%, >1%). Стекед-бар по каждому токену + сравнение по топ-20. С 📸 — миграция между корзинами («киты раздают шримпам» = классическая дистрибуция).

**13. P/H Ratio (mcap на холдера)** ✅ — $мкап/холдер vs медиана топ-20: «у DOGS $4.7 капы на холдера, у UTYA $2,514 — кто из них надут?» Простейший кросс-скрининг переоценки, считается прямо сейчас.

## IV. Индексы и скринеры (флагманские страницы)

**14. TON Quant Screener** ✅+📸 — комбо-фильтры в духе Nansen Token Screener, но бесплатно: «Score > 60 И холдеры растут 7д И цена флэт И объём ↑» → шорт-лист возможностей. Это страница, ради которой возвращаются ежедневно. База работает на текущих данных, полнота — со снапшотами.

**15. Индексы экосистемы** ✅ — TON Meme Index (mcap-weighted из наших мемов), Ecosystem TVL Index, breadth-индикатор (% токенов выше своей 7д-средней — «ширина» ралли: 18/20 растут = здоровое, 3/20 тащат индекс = хрупкое).

**16. Market Map (treemap)** ✅ — главная как карта рынка: размер плитки = mcap, цвет = 24h. Стандарт CMC/CoinGecko, на TON-экосистему — ни у кого.

**17. Кросс-токен Whale Tape** ✅ (с ключом) — единая лента крупных движений по всем топ-20: «$48K UTYA transfer → OKX» — на главной, живая. Самая залипательная фича.

## V. Что нужно для полной мощности

Снапшот-инфраструктура 📸: один GitHub Actions cron (daily), складывает JSON в репо: holders_count, TVL, volume, txns, price по топ-20. Бесплатно, без серверов, через 2 недели уже работают дивергенции (п.1, 2, 4), через месяц — полноценные тренды. Это единственный разблокирующий шаг — все 📸-фичи зависят только от него.

Приоритет по соотношению эффект/усилия:
1. **Slippage Map** (✅, день) — сразу полезно каждому трейдеру
2. **Корреляционная матрица + бета** (✅, день) — вау-эффект, уникально для TON
3. **Screener + индексы + Market Map** (✅, 2–3 дня) — флагманские страницы
4. **Снапшот-cron** (📸, полдня) — запустить как можно раньше, данные копятся только вперёд
5. **Дивергенции и CVD** (📸, после 2 недель данных) — главная альфа в долгую
